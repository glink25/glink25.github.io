## 关于

这是一个基于 VitePress 的个人博客项目。目前 VitePress 正式版尚未发布，但是目前版本已经具有一定的可用性，相比于 VuePress，我更看重 Vite 带来的快速启动和 Tree Shaking 能力，以及它的简洁和轻量化。VitePress 自带的主题已经可以满足大部分的文档需求，但是对于个人博客来说还是稍显单板，因此打造了这样一个十分简洁的主题模版。

## 使用

将本项目 clone 到本地后替换 posts 内文档即可使用。

## 配置

目前只制作了一些简单的功能，只需在`.vitepress/config.ts`中修改即可。
