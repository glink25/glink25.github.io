开启 Cent AI 助手：通用配置与进阶指南

通过接入 AI 大模型，Cent 记账 App 可以实现自动识别账单、分析消费习惯等智能功能。由于 Cent 是一个 PWA（渐进式 Web 应用），在配置时有一些特殊的注意事项，请跟随本指南完成设置。

一、 核心配置步骤

无论你使用哪家 AI 服务商（如 OpenAI、DeepSeek、智谱、Kimi 等），配置逻辑都是通用的。

1. 获取 API 信息

你需要在 AI 服务商的控制台获取三个核心参数：

  • API Key：你的专属密钥。

  • API URL：接口的基础请求地址。

  • Model Name：模型名称（例如 gpt-4o, deepseek-chat）。

2. 填写示例（以智谱 GLM 为例）

如果你想尝试国产免费额度，可以参考以下参数填写：

配置项示例值

配置名称： 我的 AI 助手

API 类型： Open AI 兼容

API URL： https://open.bigmodel.cn/api/paas/v4

API Key： 你的 sk-xxxxxxxxxxxx 可以访问智谱官网获取免费key

模型 glm-4-flash

3. Google Studio

Google 的 Gemini 模型目前提供非常慷慨的免费额度，但其配置路径略有不同。

  • 获取 Key：访问 Google AI Studio 并创建 API Key。

  • 配置参数：

    • API URL: https://generativelanguage.googleapis.com/v1beta

    • API Key: 填写你的 Gemini Key。

    • 模型: gemini-2.5-flash。

  • 注意：Google 的接口对地区有一定限制，并且完全禁止跨域，所以只能使用代理才能进行访问，请确保阅读下面的代理说明后进行配置。

二、 ⚠️ 关键：PWA 环境与跨域限制

这是配置过程中最需要注意的地方：

Cent 是一款 PWA 应用，直接运行在浏览器中。 由于浏览器的安全策略（CORS），大多数 AI 厂商（如Google、字节跳动等）默认不允许网页直接发起 API 请求。如果你直接填写厂商的官方地址，可能会遇到“网络请求失败”或“跨域拦截”的报错。

解决方案：使用 API 代理

三、 测试与保存

  1. 点击 [测试连接]：

    • 如果提示成功：说明你的代理地址、Key 和模型名称均配置无误。

    • 如果提示失败：请优先检查你的代理服务器是否正常工作，以及 API Key 是否有余额。

  2. 点击 [保存]：保存后，你就可以在账单页面使用 AI 语音或文字记账了。

  • 所有的 AI 厂商都需要代理吗？

  • 为什么我保存后 AI 没有反应？

    请确认在“模型”一栏填写的名称是否与厂商要求完全一致（注意大小写和连字符）。

  • 数据安全吗？

    Cent 会通过云同步你的配置以便多端使用，但你的 API Key 对其他协作成员是不可见的。
